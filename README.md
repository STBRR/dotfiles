## My Dotfiles 
Personal configs and setup for linux boxes.

### Setup
* Shell: zsh (jreese)
* Font: Inconsolata
* File Browser: Thunar
* Color Schemer: pywal
* Window Manager: i3-wm
* Status Bar: i3-blocks
* Search Menu: rofi
* Browser: Firefox
* Editor: Sublime Text

### Installation
* `git clone https://github.com/scktz/dotfiles.git && cd dotfiles/; sudo sh install.sh`
