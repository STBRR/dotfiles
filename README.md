## Dotfiles

A gruv-box theme for my i3.

### Setup

- Window Manager: `i3` w/ `i3-gaps`
- Terminal Emumlator: `alacritty`
- Notifications: `dunst`
- File Browser: `ranger` + `thunar`
- Launcher: `rofi`
- Status Bar: `i3blocks`
- Shell `zsh`
- Editor: `vscode` + `nano`
