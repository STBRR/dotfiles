# My Dotfiles. 

## Installation?
  * Clone the repo: `$ git clone https://github.com/stabbxr/dotfiles.git && cd dotfiles`
  * Change script permissions and execute`$ chmod +x install.sh && ./install.sh`
  * Reboot!

## Setup
  * Distro: `Ubuntu 16.04`
  * Window Manager: `i3`
  * Status Bar: `i3blocks`
  * Launcher: `rofi`
  * Terminal: `rxvt-unicode`
  * Shell: `zsh`
  * Wallpaper Manager: `feh`
  * Font: `tewi`
