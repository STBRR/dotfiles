# My Dotfiles. 

## Installation?
  * Clone Repo: `git clone https://github.com/stabbxr/dotfiles.git`
  * Run `install.sh`
  * Done :)

## Current Setup
  * Linux Distro: `Ubuntu 16.04 LTS`
  * Window Manager: `i3-gaps`
  * Bar: `i3blocks`
  * Launcher: `rofi`
  * Terminal: `xterm`
  * Shell: `bash`
  * Wallpaper Manager: `feh`
  * Editor: `subl` + `nano`
  * Terminal Font: `tewi` (thanks to lucy)
  * Video Player: `mpv`
