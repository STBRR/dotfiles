# Linux Dotfiles 

## Installation?
  * Clone: `$ git clone https://github.com/stabbxr/dotfiles.git && cd dotfiles`
  * Run Script `$ chmod +x install.sh && ./install.sh`
  * Reboot :)

## Setup
  * Distro: `Debian`
  * Window Manager: `i3wm`
  * Status Bar: `i3blocks`
  * Launcher: `rofi`
  * Terminal: `xterm`
  * Shell: `zsh`
  * Wallpaper Manager: `feh`
  * Font: `tewi`
